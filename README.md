# Ejemplo1.Fundamentos

Este proyecto contiene varios ejercicios básicos en Python para practicar conceptos fundamentales de programación.

## Estructura del Proyecto

- `xd.py`: Programa de pedido de pizza en una pizzería.
- `Ejercicio calculadora/`: Carpeta con una calculadora básica.
  - `Calculadora.py`: Programa principal de la calculadora.
  - `Funciones.py`: Funciones auxiliares para la calculadora.
  - `extenciones.py`: Archivo vacío (posiblemente para extensiones futuras).
- `Ejercicios_guia/`: Ejercicios de validación.
  - `Ejercicio_1.py`: Validador de nombre de usuario.
  - `Ejercicio_2.py`: Validador de contraseña.

## Requisitos

- Python 3.10.6 o superior

## Cómo Ejecutar

Cada archivo Python puede ejecutarse individualmente con:

```bash
python nombre_del_archivo.py
```

Por ejemplo:
- `python xd.py`
- `python Ejercicios_guia/Ejercicio_1.py`
- `python Ejercicio calculadora/Calculadora.py`

## Descripción de los Ejercicios

### Pizzería (xd.py)
Programa interactivo para ordenar pizzas con ingredientes extra, verificando el presupuesto disponible.

### Validador de Usuario (Ejercicio_1.py)
Valida que el nombre de usuario tenga entre 6 y 15 caracteres, comience con una letra y contenga solo caracteres alfanuméricos.

### Validador de Contraseña (Ejercicio_2.py)
Verifica que la contraseña tenga al menos 8 caracteres, contenga letras y números, y no tenga espacios.

### Calculadora (Ejercicio calculadora/)
Calculadora básica que permite realizar operaciones aritméticas: suma, resta, multiplicación, división, potenciación y módulo.
